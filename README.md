# SellLite v3 - Complete

This is a complete, ready-to-run SellLite v3 sample application.

## Features
- Node.js + Express backend with SQLite (better-sqlite3)
- Product listing, detail
- Admin panel (login, create product, upload image, delete)
- Cart (session-based)
- Mock payments that create orders and decrement stock
- Seed script to create admin and sample products
- Dockerfile for easy deploy

## Run locally
1. npm install
2. npm run seed
3. npm start
4. Open http://localhost:3000
